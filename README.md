# The Skill Skiller 🔨

The skill that builds skills. A smart interview, automated construction, and real eval testing.

## Install

```
npx skills add JayGPTPro/the-skill-skiller -g
```

Then just type in Claude Code:

```
/the-skill-skiller
```

## What it does

- Interviews you with questions tailored to your domain (marketing, finance, legal, code...)
- Narrows every skill to one sharp task instead of 20 mediocre ones
- Builds the complete skill and shows it for approval before saving anything
- Runs a two-layer eval: that the skill fires when it should, and that it works correctly
- Packages the result for sharing with install instructions
